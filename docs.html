Front-End Development (HTML, CSS, JavaScript)

The intern may be exposed to HTML, CSS, and JavaScript while enhancing general coding knowledge. Initially, the intern may develop a single wireframe-based front-end suitable for customer testing. This process may be iterative, requiring multiple edits and refinements before deployment as a live page. This introductory project is intended to support gradual skill development and foundational understanding of front-end workflows.

Back-End Development & DevOps

The intern may work closely with engineering teams to ensure system consistency and improve user experience as a back-end developer. Responsibilities may include participation across the full application lifecycle, including coding, debugging, deployment, and troubleshooting legacy systems. The intern may assist with implementing new endpoints, developing new back-end projects, validating code quality prior to deployment, and maintaining DevOps and cloud infrastructure. This experience is designed to strengthen back-end skills and contribute portfolio-ready work.

Weather API Web Application

The intern may assist in developing a web application that integrates real-time weather data via a weather API. Responsibilities may include researching and selecting a suitable API, integrating the API into the application, and designing a user interface to display weather data clearly and accurately. Deliverables may include a functional web application, user-friendly UI, and documentation describing the API integration process.

Python Software Engineering

In this role, the intern may support the design and implementation of software solutions addressing complex system development and integration needs. Activities may include Python coding using best practices, continuous integration workflows, automated data and metrics collection, internal feature coordination, and architectural discussions. Collaboration across teams is emphasized.

Progressive Web App (PWA) Conversion

The intern may help convert an existing website into a Progressive Web App (PWA) to improve performance, accessibility, and offline functionality. Tasks may include site analysis, PWA feature implementation, cross-device testing, and performance evaluation. Deliverables may include a fully functional PWA, a comprehensive conversion report, and a presentation demonstrating features and benefits.
